# Contact Book

## Description
This project is a Contact Book application developed using Python. It allows users to store and manage contact information using a simple menu-driven interface.

## Features
- Add a new contact
- View all contacts
- Search contacts by name
- Update existing contacts
- Delete contacts
- Simple command-line interface

## Technologies Used
- Python
- Visual Studio Code

## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run:

```bash
python contact_book.py
```
