# Simple Calculator

## Description
This project is a simple calculator developed using Python. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- Displays an error message for invalid choices

## Technologies Used
- Python
- Visual Studio Code

## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run the following command:

```bash
python calculator.py
```
